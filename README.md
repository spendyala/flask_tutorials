# Flask Tutorials

## About

This Flask Tutorial series uses `Python3.6`

## Prerequsites
- `python3.6`
- `virtualenv`
- `pip`

## Create virtual environment.
```
virtualenv -p `which python3` .venv
```

## Install requirements.txt
- `pip install -r requirements.txt`

## Run the server
  `python hello.py`
